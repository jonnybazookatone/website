Astronomical Software

github:pyweb

Python object oriented class to describe a webpage. Parses the page you specify and allows you to query the object with inbuilt functions.
					
github:pyservatory

A wrapper for the python code SkyCalc. The original code is in python, but is not written so nicely that I made an object oriented version.
					
github:hawki

A pipeline that is used to automatically determine if a GRB is observable from Paranal, Chile. It checks if the correct instrument is mounted and then creates all the required documentation to trigger the telescope.


Generic Software

github:backuptools

Object oriented classes of folders and usbdisks written in python. It makes it easier to write simple scripts to backup folders, when the full disk does not need to be stored.
					
github:rss

A class based on python RSS tools that allows the latest ArXiv e-print list to be downloaded. It then searches based on pre-defined user keywords.From this you can e-mail yourself or generate pages of the relevant papers of interest for each day.
